## 💼 Featured Projects

### 🩺 Wound Analysis & 3D Depth Estimation System

Developed an AI-driven wound analysis platform using deep learning models to detect and segment wound regions from clinical images. Integrated monocular depth estimation and 3D surface modeling to support wound depth and volume assessment.

### 🧬 Tissue Segmentation Platform for Biomedical Imaging

Built a deep learning-based tissue segmentation pipeline for biomedical and wound imaging. The system identifies necrosis, granulation tissue, and epithelialization using Detectron2 and semantic segmentation methods.

### 🧪 Agentic AI System for Transcriptome Analysis

Designed and deployed an Agentic AI-powered system to automate RNA-seq workflows, including cohort retrieval, dataset harmonization, differential expression analysis, pathway enrichment, and gene-level tracing.

### 🧠 Genetic Test Recommendation System

Developed an intelligent clinical recommendation platform using LangGraph, LangSmith, Writer LLM, and Neo4j Knowledge Graph to provide explainable genetic testing recommendations.
